## **1\. 화면 구성**

![][image1]

### **1-1. 상단영역**

* 상단 영역에서는 시스템 메뉴 열기, 기본속성(레이어, 색상, 선스타일, 문자스타일, 치수스타일), 속성바 & 속성창 제어, 인쇄 및 외부 프로그램 실행, 프로그램 화면 이동 및 종료 등을 수행할 수 있습니다. 

![][image2]

| NO | 기능 | 기능 설명 |
| :---: | :----- | :----- |
| 1 | 시스템 메뉴창 열기 | 파일 열기 및 생성, 옵션 변경, 프로그램 정보 보기 |
| 2 | 파일 목록 제어 | 열려있는 파일 목록 확인 및 파일 생성, 파일 닫기  |
| 3 | 공통속성  | 레이어, 색상, 선스타일, 가중치, 스케일, 문자스타일, 치수 스타일을 조작 |
| 4 | 속성바 | 현재 선택된 객체 또는 객체 그룹에 대한 속성 보기 및 조작 |
| 5 | 속성바 & 속성창 열기,닫기 | 속성바 또는 속성창 열기 또는 닫기 |
| 6 | 인쇄하기 & 도면관리프로그램 열기 | 도면 인쇄(Plot) 또는 도면 관리프로그램 열기 |
| 7 | 화면 이동, 프로그램 최소화, 닫기 | 프로그램을 종료하거나, 화면 이동 조작 |

### **1-2. 도면 표시**

* 도면이 표시되는 영역입니다.   
* 오른쪽 하단의 레이아웃 탭을 통해, 다른 레이아웃으로 이동할 수 있습니다.   
* 오른쪽 상단의 뷰큐브를 통해, 화면을 회전시킬 수 있습니다. 

![][image3]

### **1-3. 속성창**

* 선택된 객체의 속성이 표시되는 영역입니다. 아무 객체도 선택되어있지 않는 경우, 기본 속성(레이어, 색상, 문자속성 등등)이 조작가능합니다.

<Image src={image4} alt="" width={300} />

### **1-4. 하단 영역**

* 하단 영역에서는 간단한 ON/OFF 옵션 및 명령어 입력 및 실행, 명령어 전체보기 등을 수행할 수 있습니다.

![][image5]

| NO | 기능 | 기능 설명 |
| :---: | ------- | ----- |
| 1 | 보조기능 전체보기 | 명령어 전체보기 화면 중, 보조기능 열기 |
| 2 | 옵션 켜기 / 끄기 | 스냅, 그리드 등 옵션을 켜고 끄기 (보조기능 화면에서 해당 목록으로 드래그&드롭으로 목록 수정 가능) |
| 3 | 명령어 창 | 명령어 입력창 |
| 4 | 명령어 바로 실행 | 명령어 바로가기 실행 (명령어 전체보기에서 해당 목록으로 드래그&드롭으로 목록 수정 가능) |
| 5 | 명령어 전체보기 | 속성바 또는 속성창 열기 또는 닫기 |
