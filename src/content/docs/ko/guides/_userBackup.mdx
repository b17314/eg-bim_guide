* 작업화면 색상, 커서 설정, 명령어 팝업창 옵션 등 사용자 설정을 백업하고 복원하는 기능입니다.  
* 상단바 좌측의 로고 버튼을 클릭했을 때 뜨는 메뉴바의 사용자 설정 탭에서 사용자 설정을 백업하고, 백업했던 설정을 복원할 수 있습니다.

* 사용자 설정 탭  
- 저장하기: 현재 사용자 설정을 백업합니다.  
- 불러오기: 백업되어있는 사용자 설정을 불러와 적용시킵니다.

## **1\. 사용자 설정 백업하기**

1\) 상단바 좌측의 EG-BIM 로고 버튼을 클릭합니다.  
2\) 사용자 설정 탭의 “저장하기” 버튼을 클릭합니다.  
3\) “사용자 설정을 내보냈습니다.” 메세지 박스가 뜨고 현재 사용자 설정 백업이 완료됩니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**]() 

## **2\. 사용자 설정 복원하기**

1\) 상단바 좌측의 EG-BIM 로고 버튼을 클릭합니다.  
2\) 메뉴바에서 사용자 설정 탭의 “불러오기” 버튼을 클릭합니다.  
3\) “사용자 설정을 가져왔습니다.” 메세지 박스가 뜨고 백업했던 사용자 설정이 불러와집니다.
