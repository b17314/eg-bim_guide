* 이지빔을 사용하기 위해서는 사용자를 인증해야 합니다.   
* 이지빔은 오프라인 로그인 기능을 지원하며, 최종 온라인 인증된 시간으로부터 30일 또는 라이센스 유효 기간 중 더 짧은 시간을 기준으로 인증이 수행됩니다.

## **1\. 로그인 하기**

1\) 첫 실행 또는 로그아웃 후, 이지빔을 실행하면 로그인 창이 발생합니다.  
![][image1]  
2\) 아이디와 비밀번호를 입력 후, 로그인 버튼을 클릭합니다.  
\* 아이디가 없는 경우, 회원 가입 버튼을 통해 홈페이지에서 회원가입을 진행해야 합니다.  
\* 비밀번호가 기억나지 않는 경우, 비밀번호 재설정을 통해 비밀번호를 복구할 수 있습니다.

## **2\. 로그아웃 하기**

1\) 모든 도면을 종료합니다.  
2\) 이지빔 좌측 상단에 ![image alt emoji][image2] (이지빔 로고) 버튼을 클릭합니다.  
3\) 홈메뉴 중 ‘궁금하면’ 의 ‘S/W 정보’ 를 클릭합니다.   
![][image3]  
4\) ‘About EG-BIM’ 화면 하단에 ‘LOGOUT’ 버튼을 클릭합니다.  
![][image5]  
\* 도면이 활성화 되있는 경우, 로그아웃을 할 수 없습니다. 모든 도면을 종료 후, 수행하시기 바랍니다.

## **3\. 현재 로그인 한 사용자 및 라이센스 유효일 확인하기** 

1\) 이지빔 좌측 상단에 ![image alt emoji][image2] 버튼을 클릭합니다.  
2\) 홈메뉴 중 ‘궁금하면’ 의 ‘S/W 정보’ 를 클릭합니다.   
![][image3]  
3\) ‘About EG-BIM’ 화면에서 사용자 정보 및 라이센스 유효일을 확인할 수 있습니다.  
![][image4]

\* 내부 사용자의 경우, 별도의 날짜 표기되지 않습니다.
