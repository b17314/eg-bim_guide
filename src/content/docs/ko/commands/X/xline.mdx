* 양방향 무한 선을 생성합니다.  
* 시작점을 기준으로 양쪽 방향으로 무한히 향하는 직선을 그립니다.



#### 양방향 무한선 그리기 옵션 키워드  

![][image1]

| 옵션 | 기능 설명 |
| :--- | :----- |
| Bisect | vertex point와 angle start point을 지정하여 angle end point과 angle start point간의 각도의 절반에 해당하는 각도(양방향 무한선과 angle start point간의 각도 \<= 90)를 가지는 양방향 무한선을 그립니다. |
| Horizontal | 수평 방향의 양방향 무한선을 그립니다.  |
| Vertical | 수직 방향의 양방향 무한선을 그립니다. |
| Angle | Reference를 클릭하여 다른 객체의 Angle을 참조하거나, 지정된 각도로 양방향 무한선을 그립니다.   |
| Offset | 선택한 직선과 평행하고 입력한 Offset만큼의 거리를 가지는 양방향 무한선을 그립니다. |

#### Offset 키워드 입력 후 평행 양방향 무한선 그리기 키워드  

![][image2]

| 옵션 | 기능 설명 |
| :--- | :----- |
| Through Point | 직선 객체와 평행할 양방향 무한선의 위치를 마우스로 자유롭게 지정합니다.   |
| Through | 화면 상에 마우스로 두 점을 지정하고 그 두 점 사이의 거리를 직선 객체와 평행할 양방향 무한선 간의 거리로 지정합니다. |
| 키워드 입력 없이 거리 입력 | 직선 객체와 평행할 양방향 무한선 간의 거리를 지정하고 이후 마우스로 직선 객체를 기준으로 양방향 무한선의 상하좌우 위치를 지정합니다. |

<br/>

## **1\. 양방향 무한선 그리기 방법**

1\) “XLINE” 명령어를 실행합니다.  
2\) 화면 상에 마우스로 무한선의 시작점을 지정합니다.  
3\) 화면 상에 마우스로 무한선의 Direction을 지정하면 양방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 단방향 무한선 그리기를 종료합니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/ZMrej7K6ZqQ)

## **2\. 두 점을 이용한 양방향 무한선 그리기 방법**

1\) “XLINE” 명령어를 실행합니다.  
2\) Bisect 옵션을 입력합니다.  
2\) 화면 상에 마우스로 무한선의 시작점을 지정합니다.  
3\) 화면 상에 마우스로 angle start point와 angle end point를 지정하면 양방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 양방향 무한선 그리기를 종료합니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/nJnh48n6Hs0)

## **3\. 수직/수평 무한선 그리기 방법**

1\) “XLINE” 명령어를 실행합니다.  
2\) Horizontal 키워드(수평) 또는 Vertical 키워드(수직)를 입력합니다.  
3\) 마우스로 화면 상에 시작점을 지정하면 수평 또는 수직의 양방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 양방향 무한선 그리기를 종료합니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/_DBF5iyw57Y)

## **4\. 특정 각도를 가진 양방향 무한선 그리기 방법**

1\) “XLINE” 명령어를 실행합니다.  
2\) Angle 키워드를 입력합니다.  
3\) Reference를 클릭하여 객체를 참조하거나 원하는 각도를 입력하고 ENTER 키를 누릅니다.  
3\) 마우스로 화면 상에 시작점을 지정하면 입력한 각도를 가진 양방향 무한선이 그려집니다.  
4\) ENTER 또는 ESC 키를 눌러 양방향 무한선 그리기를 종료합니다.
