* 도면 내에서 선택한 한 점의 좌표(X, Y, Z) 값을 확인하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “ID” 입력합니다.  
2\) 도면 내에서 좌표를 확인하고 싶은 점을 클릭합니다.  
3\) 명령창에 해당 점의 좌표가 출력됩니다.
