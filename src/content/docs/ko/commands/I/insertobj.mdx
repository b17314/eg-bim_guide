* 현재 도면에 외부 응용프로그램에서 작성된 객체(OLE 객체)를 삽입하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “INSERTOBJ” 명령어를 실행합니다.  
2\) 객체 삽입 대화상자에서 원하는 옵션을 선택합니다.  
![][image2]  
3\) 확인을 누르면 객체가 도면에 삽입됩니다.  
    참고 : 삽입된 객체를 더블클릭하면 원래 응용프로그램에서 편집할 수 있으며, 수정 내용은 도면에도  
              반영됩니다.
