* 도면에서 선택한 객체의 색상(Color) 정보를 조회하고, 색상별 Color Index 및 True Color 값을 확인할 수 있는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “INQUIRECOLOR” 입력합니다.  
2\) 색상 조회 대화상자가 나타납니다.  
![][image2]  
3\) 필요한 옵션을 선택합니다.  
4\) 객체를 선택하면 색상 목록에 정보가 표시됩니다.
