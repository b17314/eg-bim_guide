### 스플라인 그리기 옵션 키워드  

![][image2]

| 옵션 | 설명 |
| :----- | :-----|
| Close | 스플라인을 닫습니다.  &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; |
| Fit tolerance | 곡선의 tolerance를 설정합니다. |

<br/>

## **1\. 스플라인 그리는 방법**

1\) “SPLINE” 명령어를 실행합니다.  
2\) 스플라인의 first point, second point과 Next Point들을 지정합니다.  
3\) 현재 그리고 있는 스플라인의 Fit tolerance를 변경하고자 하는 경우 Fit tolerance 키워드를 입력하고 원하는 값을 입력합니다.  
4\) 스플라인 그리기를 종료하고자 하는 경우 ENTER 키를 누르고, starting tangent point와 ending point의 tangent를 설정합니다.  
5\) tangent 설정이 완료되면 명령어가 종료됩니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/rhbwh0Btgrw)

## **2\. 닫힌 스플라인 그리는 방법**

1\) “SPLINE” 명령어를 실행합니다.  
2\) 스플라인의 first point, second point과 Next Point을 지정합니다.  
3\) Close 키워드를 입력하고 tangent를 입력합니다.  
4\) tangent 설정이 완료되면 명령어가 종료됩니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/Z8ew1O4lvQA

## **3\. 2D 폴리선을 스플라인으로 변환하는 방법**

1\) “SPLINE” 명령어를 실행합니다.  
2\) Entity 키워드를 입력합니다.  
3\) 스플라인으로 변환할 spline fitted polyline 객체를 선택합니다.  
4\) ENTER 키를 누르면 spline fitted polyline이 스플라인으로 변환됩니다.
