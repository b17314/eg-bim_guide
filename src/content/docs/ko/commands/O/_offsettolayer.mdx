* 선택한 레이어로 OFFSET 명령을 수행합니다.  
  ![][image1]  
* 현재 레이어로 변경 체크 시 오프셋이 완료된 후 생성된 새 객체를 현재 레이어로 이동시킵니다.  
* 레이어 선택 시 오프셋이 완료된 후 생성된 새 객체를 선택한 레이어로 이동시킵니다.  
* 일반 Offset : 선택한 객체를 지정 거리만큼 한쪽 방향으로 평행 복제  
* 증분 Offset : 여러 번 오프셋 시, 이전 오프셋 거리의 배수(증가값)로 연속 생성  
* 양쪽 Offset : 선택 객체 기준 좌우 대칭 방향으로 동시에 오프셋 생성  
* 여러 번 Offset : 한 번 명령으로 여러 간격의 오프셋 라인을 연속 생성  
* 양쪽 증분 Offset : 좌우 대칭 \+ 각 방향마다 증분 적용  
* 


## **1\. 사용 방법**

1\) 명령창에 “OFFSETTOLAYER” 명령어를 입력합니다.  
2\) OFFSETTOLAYER 대화상자에서 OFFSET 조건을 설정합니다.  
3\) 설정 완료 후 확인 버튼을 클릭합니다.  
4\) 대화상자의 레이어 항목에서 선택한 레이어에 offset을 통해 이동시킨 객체가 생성됩니다.  
5\) 설정된 Offset 방식에 따라 OFFSET 명령을 수행합니다.
