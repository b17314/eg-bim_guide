* 도면에서 중복되거나 불필요하게 겹쳐 있는 객체를 자동으로 찾아서 삭제하거나 병합하여 정리하는 기능입니다.

![][image1]

## **1\. 중복 객체 삭제**

1\) “OVERKILL” 명령어를 실행합니다.  
2\) 중복 여부를 검사할 객체를 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) “객체 정리(Overkill)” 옵션 대화상자가 나타납니다.  
![][image2]  
5\) 원하는 옵션을 설정 후 “확인”을 클릭합니다.  
6\) 중복 객체가 삭제되고, 정리된 객체만 남습니다.
