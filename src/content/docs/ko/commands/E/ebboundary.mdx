* 선택한 객체를 기준으로 내부 또는 외부 경계 영역을 잘라내는 명령어입니다.

![][image1]

## **1\. 객체 외부 / 내부 잘라내기**

![][image2]  
1\) “EBBOUNDARY” 명령어를 실행합니다.  
2\) 경계 내부 또는 경계 외부를 선택 합니다.

- 경계 내부 : 선택한 객체를 기준으로 내부 영역의 선을 잘라냅니다.  
- 경계 외부 : 선택한 객체를 기준으로 외부 영역의 선을 잘라냅니다.

3\) 경계에서 잘라낼 수 없는 객체(블록, 치수, 해치)에 대해 처리 방식을 선택 합니다.

- 무시 : 해당 객체는 잘라내기 대상에서 제외 됩니다.  
- 폭파 후 처리 : 객체를 분해한 뒤 잘라내기를 수행합니다.

4\) 확인 버튼을 누릅니다.  
5\) 잘라낼 대상을 선택 합니다.  
6\) ESC 키를 눌러 종료합니다.
