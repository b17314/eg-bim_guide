* 도면 내 객체의 좌표를 추출하여 엑세 파일로 내보내거나, 좌표 정보를 도면 상에 문자로 표시하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “EXPORTCTE” 입력합니다.  
2\) 엑셀 창이 자동으로 실행되며, 좌표가 입력될 셀 위치를 선택합니다.  
3\) 표시된 설정 창에서 필요한 옵션을 설정합니다.  
![][image2]  
3\) 도면에서 좌표를 추출할 지점을 순서대로 클릭합니다.  
4\) 선택이 완료되면 Enter 키를 누릅니다.  
5\) 선택된 좌표가 엑셀 파일(.xls 또는 .xlsx)로 자동 내보내집니다.  
![][image3]
