* 타원을 생성합니다.  
* 새로운 타원을 생성하거나 타원의 일부인 호를 생성할 수 있습니다.

![][image1]

* 타원 그리기 옵션 키워드  
- Arc: 타원을 그리고 해당 타원이 남겨질 부분을 지정해 호를 그립니다.  
- Center: 중심점과 종점을 지정하여 타원을 생성합니다.

## **1\. 타원을 그리는 방법**

1\) “ELLIPSE” 명령어를 실행합니다.  
2\) 화면 상에 마우스로 타원 축의 두 종점을 지정합니다.  
3\) 단축의 길이를 지정하여 타원을 생성합니다. (2가지 방법)  
&nbsp;&nbsp;&nbsp;&nbsp;3-1) 화면 상에 마우스로 Other Axis를 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;3-2)  Rotation 키워드를 입력해 Rotation around major axis를 지정합니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/f6rgRr0UNm0)

## **2\. 타원에서 파생된 호를 그리는 방법**

1\) “ELLIPSE” 명령어를 실행합니다.  
2\) Arc 키워드를 입력합니다.  
3\) 타원 축의 양 종점을 지정합니다.  
4\) 화면 상에 마우스로 Other Axis를 지정합니다.  
5\) 남겨질 부분을 지정하면 호가 그려집니다. (2가지 방법)  
&nbsp;&nbsp;&nbsp;&nbsp;5-1) start angle of arc, end angle을 화면 상에 마우스로 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;5-2) Parameter 키워드를 입력해 start, end parameter를 입력합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;5-3) start angle 또는 start parameter 입력 이후 Included 키워드를 입력, 포함될 각도를 입력합니다.
