![][image1]

## **1\. 객체 외부 잘라내기**

1)“EXTENDTRIM” 명령어를 실행합니다.  
2\) 기준이 되는 객체를 선택합니다.  
3\) 삽입할 위치 좌표를 입력 합니다.  
4\) 엔터키나 스페이스바를 눌러 종료합니다.
