* 여러 줄의 텍스트를 작성하고 다양한 서식을 적용할 수 있는 기능입니다.

![][image1]

## **1\. 여러 줄 텍스트 작성**

1\) “MTEXT” 또는 “MT” 명령어를 실행합니다.  
2\) 화면에서 텍스트를 입력할 영역(사각형 창)을 지정합니다.

- 첫 번째 점: 텍스트 상자의 왼쪽 위 모서리  
- 두 번째 점: 텍스트 상자의 오른쪽 아래 모서리

3\) 지정된 영역 안에 나타나는 텍스트 편집기에 글을 입력합니다.  
4\) 텍스트 입력을 완료한 뒤, ESC 키를 누르거나 Ctrl \+ Enter를 눌러 종료합니다.
