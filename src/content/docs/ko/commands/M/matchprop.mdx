* 객체의 속성을 다른 객체에 복사합니다.  
* SETTING 원본 객체 선택시 복사할 속성 선택 키워드  
  ![][image1]  
  \- Color : 원본 객체의 색상을 복사합니다..  
  \- Layer : 원본 객체가 속한 레이어를 대상 객체에 적용합니다.  
  \- Linetype : 원본 객체의 선 종류를 복사합니다.  
  \- Linetype Scale : 원본 객체의 선 종류가 반복되는 간격을 복사합니다.  
  \- Lineweight : 원본 객체의 선 가중치를 복사합니다.  
  \- Transparency : 원본 객체의 투명도를 복사합니다.  
  \- Thickness : 원본 객체의 두깨를 복사합니다.  
  \- Print Style : 원본 객체의 출력 스타일을 복사합니다.  
  \- Height / Scale : 원본 객체의 크기 속성을 복사합니다.  
  \- Angle : 원본 객체의 각도 를 복사합니다.  
  \- Dimension : 원본 객체의 치수를 복사합니다.  
  \- Viewport : 뷰포트의 특성을 복사합니다.  
  \- Multileader : 지시선 속성을 복사합니다.  
  \- Polyline : 폴리선의 속성을 복사합니다.  
  \- Hatch : 해치의 속성을 복사합니다.  
  \- Block Reference : 블록 객체의 속성을 복사합니다.  
  \- Text : 문자의 속성을 복사합니다.  
  \- Table : 표 객체의 속성을 복사합니다.

## **1\. 객체 복사하기** 
1\) MATCHPROP 명령어를 실행하여 원본 객체를 선택합니다. 
![][image2] 
2\) 속성을 복사할 대상 객체를 선택하면 원본의 속성이 대상 객체에 적용됩니다.
![][image3]
