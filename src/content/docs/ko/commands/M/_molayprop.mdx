* 선택한 객체의 레이어 속성을 일괄적으로 변경할 수 있는 기능입니다. 여러 객체가 서로 다른 레이어에 있어도, 지정한 레이어 속성으로 통일할 수 있습니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “MOLAYPROP” 명령어를 입력합니다.  
2\) 속성을 변경할 객체들을 선택합니다.  
3\) MOLAYPROP 대화상자에서 변경할 레이어 속성을 지정합니다.  
![][image2]  
4\) 확인 버튼을 클릭하면 선택한 객체들의 레이어 속성이 지정된 값으로 변경됩니다.
