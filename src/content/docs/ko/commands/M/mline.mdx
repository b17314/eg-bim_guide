## **1\. 여러줄 그리기 방법**

1\) Start Point 클릭, Next Point 클릭  
2\) 원하는 지점으로 Next Point를 클릭하다가 여러줄을 다 그렸으면 스페이스바or엔터키를 누릅니다.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;-> 원하는 지점으로 Next Point를 클릭하다가 \[Close/Undo\]를 수행할 수 있습니다.
<br/>&nbsp;&nbsp;&nbsp;&nbsp;2-1) 명령어창에 Close를 클릭하거나 입력한 경우 Start Point를 기준으로 닫힙니다.  
&nbsp;&nbsp;&nbsp;&nbsp;2-2) 명령어창에 Undo를 클릭하거나 입력한 경우 Next Point를 그리기 직전으로 돌아갑니다.  
&nbsp;&nbsp;&nbsp;&nbsp;2-3) 원하는 지점으로 Next Point를 클릭하다가 전체취소를 원할 경우 \> ESC 키를 누릅니다.
