* 현재 작업 중인 도면 공간(출력용 공간) 에서 모델 공간(작업용 공간) 으로 전환하는 명령입니다.

![][image1]

## **1\. 사용 방법**

1\) Layout 탭으로 이동합니다.  
2\) 명령창에 “MSPACE” 명령어를 입력합니다.  
3\) 엔터 키를 누릅니다.  
4\) 현재 작업 공간이 모델 공간 으로 전환됩니다.  
    참고 : 도면 공간으로 돌아가려면 “PSPACE” 명령을 사용합니다.
