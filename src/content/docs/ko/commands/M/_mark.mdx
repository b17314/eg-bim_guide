* 클릭한 지점에 철근 마킹을 삽입하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “MARK” 를 입력합니다.  
2\) 명령창 안내에 따라 철근 마크를 삽입할 위치를 클릭합니다.  
3\) 자동으로 기본 철근 마크 블록이 해당 위치에 생성됩니다.
