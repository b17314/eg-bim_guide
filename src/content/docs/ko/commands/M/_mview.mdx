* 도면 공간(Layout Space) 에서 뷰포트를 생성하거나 관리하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) Layout 탭으로 이동합니다.  
2\) 명령창에 “MVIEW” 입력합니다.  
3\) 명령창에 표시되는 옵션 중 하나를 선택합니다.

| 옵션 | 설명 |
| :---- | :---- |
| ON/OFF | 선택한 뷰포트를 활성화(ON) 또는 비활성화(OFF) 합니다. |
| Fit	 | 현재 도면 영역에 맞게 자동으로 뷰포트를 확대·축소합니다. |
| Lock	 | 뷰포트 배율을 고정(ON) 하거나 해제(OFF) 합니다. |
| Entity	 | 기존 객체(예: 사각형, 원 등)를 뷰포트로 전환합니다. |
| Polygonal	 | 다각형 형태의 뷰포트를 새로 생성합니다. |
