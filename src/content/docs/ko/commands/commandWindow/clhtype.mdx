* 선택길이 만큼 선꼴을 변경합니다.

![][image1]  
1\) 직선(LINE) 객체를 선택합니다.  
2\) HIDDEN 선이 시작할 지점을 클릭하거나 거리를 입력합니다.  
3\) HIDDEN 선이 끝날 지점(두번째 점)을 클릭하거나 거리를 입력합니다.  
![][image2]  
4\) 선스타일의 Scale 값을 입력합니다.
