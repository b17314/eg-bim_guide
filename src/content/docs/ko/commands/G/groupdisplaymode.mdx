* 도면 화면에서 그룹의 경계 상자를 표시할지 여부를 제어하는 기능입니다.  
* 이 설정은 그룹 자체에는 영향을 주지 않고, 화면 표시만 달라집니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “GROUPDISPLAYMODE” 명령어를 입력합니다.  
2\) 원하는 값을 입력합니다.

- 0 : 그룹 선택 시 경계 상자를 표시하지 않음  
- 1 : 그룹 선택 시 단순한 점선 사각형 경계 상자 표시  
- 2 : 그룹 선택 시 보다 두드러진 사각형 경계 상자 표시  
- 3 : 그룹 선택 시 경계 상자와 함께 객체의 선택 핸들도 강조 표시
