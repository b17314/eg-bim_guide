* 여러 객체를 하나의 그룹 으로 묶어 관리할 수 있는 기능입니다.  
* 그룹은 블록처럼 정의 파일을 만들지 않고, 단순히 선택/편집을 쉽게 하기 위해 묶어 놓은 개념입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “GROUP” 명령어를 실행합니다.  
2\) 그룹할 객체를 선택 합니다.  
3\) 그룹을 구분하기 위한 이름을 지정합니다.  
4\) 엔터 또는 스페이스 키를 누릅니다.  
5\) 선택한 객체들이 하나의 그룹으로 묶입니다.  
![][image2]
