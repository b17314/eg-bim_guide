* 지정한 기준점에 맞추어 자동으로 정렬해 주는 기능입니다.

![][image1]

## **1\. 텍스트 정렬**

1\) “ARTEXT” 명령어를 실행합니다.  
2\) “S” 를 입력하고 ARTEXT 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 정렬할 여러 개의 텍스트 객체를 선택합니다.  
4\) 기준점(정렬 기준이 될 점)을 지정합니다.  
5\) 모든 텍스트가 지정한 기준점에 따라 자동으로 정렬됩니다.  
![][image3]
