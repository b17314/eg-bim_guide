* 도면에서 각도(Angle) 단위 형식을 설정하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “AUNITS” 를 입력합니다.  
2\) 명령창 안내에 따라 0\~4 사이의 숫자를 입력합니다.

| 입력값 | 각도 형식 | 예시 | 설명 |
| :---: | :---: | :---: | ----- |
| 0 | 소수(Decimal Degrees) | 45.0000 | 일반적인 도 단위(°)로 표시됨 |
| 1 | 도/분/초 (Deg/Min/Sec) | 45°30'20" | 도 단위 \+ 분/초 단위로 표시 |
| 2 | 그라드(Grads) | 50.0000g | 1바퀴 \= 400 grad, 측량용 단위 |
| 3 | 라디안(Radians) | 0.7854r | 수학/공학용 라디안 단위 |
| 4 | 서베이어(Surveyor’s Units) | N 45°30'20" E | 방향(북·남·동·서) 포함 형식, 측량용에 사용 |

3\) 선택한 번호에 해당하는 형식으로 각도 표시 단위가 변경됩니다.
