* 선택한 지점에 수치 연산 결과를 문자로 표시하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “ADDTEXTVALUE” 입력합니다.  
2\) 원하는 연산 종류를 선택합니다.

- A : 더하기(Add)  
- S : 빼기(Subtract)  
- M : 곱하기(Multiply)  
- D : 나누기(Divide)

3\) 연산할 기준값을 입력합니다.  
4\) 도면에서 숫자가 포함된 텍스트 객체를 선택합니다.  (예: “25” 라는 TEXT 객체 선택)  
5\) 명령어가 자동으로 연산을 수행하고,  결과값이 새 텍스트로 도면에 표시됩니다.
