* 호를 생성합니다.  
* 호의 속성 값을 설정하여 ARC를 그릴 수 있습니다.

### 옵션 1 : ARC의 설정 옵션 키워드  

![][image1]

| 옵션 | 기능 설명 |
| :--- | :----- |
| Center | 중심점을 지정한 후, 시작점과 끝점을 마우스로 클릭하여 ARC를 생성합니다. |
| Follow | 이전 객체의 끝점이 호의 시작점이 됩니다. 끝점만 지정하여 ARC를 생성합니다. |

### 옵션 2 :  ARC를 실행하여 시작점(start point)을 지정한 후 옵션  

![][image2]

| 옵션 | 기능 설명 |
| :--- | :----- |
| Angle | 호의 각도를 설정하여 (1)호의 중심점을 지정하거나 (2)끝점을 지정할 수 있습니다.   |
| Center | 호의 중심점을 설정하여 (1)각도를 지정하거나 (2)현의 길이를 지정할 수 있습니다.  |
| Direction |  호의 방향을 설정하여 (1)시작점을 기준으로 끝점을 지정하거나, (2)각도를 설정하여 끝점을 지정할 수 있습니다.  |
| End | 호의 끝점을 설정하여 (1)각도를 지정하거나, (2)방향을 지정하거나, (3)반지름을 지정할 수 있습니다. |
| Radius | 호의 반지름을 설정하여 (1)포함된 각도를 지정후 방향을 지정하거나, (2)끝점을 지정할 수 있습니다.|




## **1\. 호 그리는 방법(3점 지정 : 기본 옵션)**

1\) “ARC” 명령어를 실행합니다.  
2\) 시작점, 두번째 점, 끝점을 지정하여 호를 생성합니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/yNbpane28ow)

## **2\. 시작점, 각도로 호 그리는 방법**

1\) “ARC” 명령어를 실행합니다.  
2\) 시작점을 지정 \> Angle 옵션 클릭 \> 각도를 설정합니다.  
2-1) Center 옵션을 클릭하고 중심점을 지정하여 호를 생성합니다.  
2-2) 끝점을 지정하여 호를 생성합니다.(default)

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/AFmTzT-li4U)

## **3\. 시작점, 중심점으로 호 그리는 방법**

1\) “ARC” 명령어를 실행합니다.  
2\) 시작점을 지정 \> Center 옵션 클릭 \> 중심점을 설정합니다.  
2-1) Angle 옵션을 클릭하고 각도를 지정하여 호를 생성합니다.  
2-2) Length of chord 옵션을 클릭하고  현의 길이를 지정하여 호를 생성합니다.  
2-3) 끝점을 지정하여 호를 생성합니다.(default)

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/LNWb7FJ1Otw)

## **4\. 시작점, 방향, 끝점으로 호 그리는 방법**

1\) “ARC” 명령어를 실행합니다.  
2\) 시작점을 지정 \> Direction 옵션을 클릭합니다.  
2-1) Follow 옵션을 클릭하고 끝점을 지정하여 호를 생성합니다.  
2-2) 호의 초기 방향 각도를 지정하고 끝점을 지정하여 호를 생성합니다.(default)

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/SKYBst2ia2I)

## **5\. 시작점, 끝점으로 호 그리는 방법**

1\) “ARC” 명령어를 실행합니다.  
2\) 시작점을 지정 \> End 옵션 클릭 \> 끝점을 설정합니다.  
2-1) Angle 옵션을 클릭하고 각도를 지정하여 호를 생성합니다.  
2-2) Direction 옵션을 클릭하고 시작 방향을 지정하여 호를 생성합니다.  
2-3) Radius 옵션을 클릭하고 반지름을 지정하여 호를 생성합니다.  
2-4) 중심점을 지정하여 호를 생성합니다.(default)

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/PKIEil8AB4g)

## **6\. 시작점, 반지름으로 호 그리는 방법**

1\) “ARC” 명령어를 실행합니다.  
2\) 시작점을 지정 \> Radius 옵션 클릭 \> 반지름을 설정합니다.  
2-1) Included angle 옵션 클릭 \> 각도 지정 \>  방향을 지정하여 호를 생성합니다.  
2-2) 끝점을 지정하여 호를 생성합니다.(default)  

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/RHK14-e_rCg)
