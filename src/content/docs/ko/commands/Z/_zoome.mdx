* 도면에 존재하는 모든 객체를 화면의 경계에 딱 맞게 확대 또는 축소하여 표시합니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “ZOOME” 명령어를 입력합니다.  
2\) 엔터 키를 누릅니다.  
3\) 화면이 자동으로 조정되며, 도면 내의 모든 객체가 화면에 전체 표시 된다.
