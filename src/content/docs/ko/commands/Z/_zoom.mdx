* 도면의 표시 범위를 조정하여 화면을 확대하거나 축소하는 기능을 수행합니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “ZOOM” 명령어를 입력합니다.  
2\) 엔터 키를 누르면 다음과 같은 옵션이 표시됩니다.  
3\) 원하는 옵션을 선택하여 화면 배율을 조정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| In (확대) | 현재 뷰의 중심을 기준으로 화면을 2배 확대합니다. |
| Out (축소) | 현재 뷰의 중심을 기준으로 화면을 0.5배 축소합니다. |
| All (전체) | 도면 한계 전체를 화면에 표시하며, Limits가 없으면 Extents와 동일하게 작동합니다. |
| Center (중심) | 지정한 점을 중심으로 화면을 재설정하고, 표시할 폭을 입력하여 확대/축소를 조정합니다. |
| Dynamic (동적) | 사각형 프레임을 사용하여 확대할 영역을 시각적으로 미리 지정하고 조정합니다. |
| Extents (범위) | 도면 내의 모든 객체가 화면에 꽉 차게 표시되도록 확대 또는 축소합니다. |
| Left (왼쪽 이동) | 화면을 왼쪽으로 이동합니다. |
| Previous (이전) | 직전 확대/축소 상태로 되돌립니다. |
| Right (오른쪽 이동) | 화면을 오른쪽으로 이동합니다. |
| Window (창) | 마우스로 두 점을 지정하여 해당 영역만 확대 표시합니다. |
| Entity (객체) | 선택한 객체를 기준으로 화면을 확대하여 해당 객체가 꽉 차게 보이도록 합니다. |
| Scale (배율) | 확대/축소 배율을 직접 입력합니다. (예: 2X \= 2배 확대, 0.5X \= 절반 축소, 1XP \= 1:1 출력 비율) |
