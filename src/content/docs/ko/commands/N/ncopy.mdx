* 내부에 포함된 객체를 선택하여 현재 도면으로 직접 복사하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “NCOPY” 명령어를 실행합니다.  
2\) 객체를 선택할 때, 블록이나 Xref 내부의 원하는 요소를 클릭합니다.  
3\) 선택한 객체가 현재 도면 공간에 복사됩니다.
