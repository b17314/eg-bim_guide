* 3D 폴리선을 생성합니다.

![][image1]

* 3D 폴리선의 시작점 지정 옵션 키워드  
- Follow 또는 ENTER 키: last point를 3D 폴리선의 시작점으로 지정  
- 화면 상에 마우스로 지정

![][image2]

## **1\. 3D 폴리선 그리기**

1\) “3DPOLY” 명령어를 실행합니다.  
2\) Start point를 마우스로 클릭합니다.  
3\) Next point를 마우스로 클릭합니다.  
4\) Undo 옵션 키워드를 사용할 수 있습니다.  
&nbsp;4-1) 사용을 원하면 Undo 키워드를 입력해 이전으로 돌아갑니다.  
5\) 3번째 Next point를 마우스로 클릭할 경우 Close 키워드나 Undo 키워드를 사용할 수 있습니다.  
&nbsp;5-1) Close 키워드를 사용할 경우 처음 Start point를 기준으로 객체를 닫습니다.(명령어 종료)  
&nbsp;5-2) Undo 키워드를 사용하면 이전으로 돌아갑니다.  
6\) ESC키나 Space키를 입력해 명령어를 종료합니다.  

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/5gfnEL3CAHY) 

## **2\. 3D 폴리선을 그리다가 이전으로 돌아가기**
1\) “3DPOLY” 명령어를 실행합니다.
2\) Start point를 마우스로 클릭합니다.
3\) Next point를 마우스로 클릭합니다.
4\) Undo 키워드를 입력하여 이전으로 돌아갑니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/L_Sso4MRlwI) 

## **3\. 3D 폴리선을 그리다가 객체 닫기**
1\) “3DPOLY” 명령어를 실행합니다.
2\) Start point를 마우스로 클릭합니다.
3\) Next point를 마우스로 클릭합니다.
4\) Close 키워드를 입력하여 시작점을 기준으로 객체를 닫습니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/hT9huG4RGHk)
