* 현재 선택된 뷰포트를 최대화 하여 전체 화면으로 확장하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) Layout 탭으로 이동한다.  
2\) 명령창에 “VPMAX” 를 입력합니다.  
3\) 엔터 키를 누릅니다.  
4\) 선택한 뷰포트가 전체 화면으로 확대되며, 모델 공간처럼 편집 가능합니다.  
    참고 : “VPMIN” 명령어를 입력하면 복귀합니다.
