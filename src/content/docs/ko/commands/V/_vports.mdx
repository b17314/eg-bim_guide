* 현재 작업 중인 화면을 여러 개의 뷰로 분할하거나, 도면 공간에서 사용할 뷰포트를 생성, 배치하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) Layout 탭으로 이동합니다.  
2\) 명령창에 “VPORTS” 를 입력합니다.  
3\) Viewports 대화상자가 나타납니다.  
4\) 화면 분할 방식(예: Single, Two, Three, Four 등)을 선택합니다.  
5\) 선택한 옵션에 따라 화면이 여러 개의 뷰로 나누거나, 새로운 뷰포트가 도면 공간에 생성됩니다.
