* 도면에서 특정 색상을 가진 객체만 화면에 표시하고, 다른 색상의 객체는 임시로 숨기는 기능입니다.

![][image1]

## **1\. 특정 색상 객체만 표시**

1\) “VIEWSELECTIONCOLOR” 명령어를 실행합니다.  
2\) 색상 선택 대화상자가 나타납니다.  
3\) 도면에서 표시할 객체의 색상을 선택합니다.  
4\) 엔터 또는 스페이스 키를 눌러 선택을 완료합니다.  
5\) 선택한 색상을 가진 객체만 화면에 표시되고, 다른 색상의 객체는 모두 숨겨집니다.  
    참고 : VIEWALLENTITIES 명령어를 실행하면 다시 전체 객체가 보입니다.
