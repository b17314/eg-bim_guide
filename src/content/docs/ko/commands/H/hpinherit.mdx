# **HPINHERIT (명령어)**

* 해치 특성 상속과 관련된 시스템 변수 명령어입니다.  
* 이 명령어는 해치 생성 시 해치 원점의 상속 여부를 설정합니다.  
- HPINHERIT \= OFF(0) : 해치 원점 상속을 비활성화  
- HPINHERIT \= ON(1) : 해치 원점 상속 활성화

![][image1]

## **1\. 해치 원점 상속 비활성화**

1\) “HPINHERIT” 명령어를 실행하여 0 또는 OFF를 입력합니다.  
2\) “RECTANGLE” 명령어를 실행하여 2개의 사각형을 그립니다.  
3\) 첫 번째 모서리 : 0,0 두 번째 모서리는 100,100을 입력후 엔터키를 눌러 사각형을 그립니다.  
4\) 첫 번째 모서리 : 150,0 두 번째 모서리는 250,100을 입력후 엔터키를 눌러 사각형을 그립니다.  
5\) “HATCH” 명령어를 실행하여 원하는 패턴을 선택합니다.  
6\) 해치 원점을 ‘지정된 원점’ \> ‘클릭하여 새 원점을 설정’ 버튼을 클릭합니다.  
7\) 원점 지정을 50,50을 입력 후 엔터키를 누릅니다.  
8\) 첫 번째 사각형을 선택후 ESC키로 종료합니다.  
9\) 다시 “HATCH” 명령어를 실행하여 우측하단의![][image2]특성 상속 버튼을 클릭합니다.  
10\) 첫 번째 해치를 클릭하여 특성 값을 가져오고 두 번째 사각형을 선택합니다.  
11\) 두 번째 해치가 생성되고 속성창에서 원점X,Y 좌표를 확인할 수 있습니다.(원점 상속x)

7\) 원점 지정을 50,50을 입력 후 엔터키를 누릅니다.  
8\) 첫 번째 사각형을 선택후 ESC키로 종료합니다.
