# **HYPERLINK (명령어)**

* 객체에 외부 웹사이트, 파일, 전자우편 주소 또는 같은 도면 내 특정 뷰로 연결되는 하이퍼링크를 삽입하는 기능입니다.

![][image1]

## **1\. 객체 하이퍼링크 추가**

1\) “HYPERLINK” 명령어를 실행합니다.  
2\) 하이퍼링크를 지정할 객체(선, 폴리선, 블록, 이미지 등)를 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) 하이퍼링크 대화 상자가 나타납니다.  
![][image2]

5\) 다음 중 하나를 입력 또는 선택합니다:

- 웹 주소(URL): ex) `https://www.google.com`  
- 파일 경로: ex) `C:\Project\설계도.dwg`

6\) “확인”을 클릭하여 설정을 완료합니다.  
7\) 객체 위에 마우스를 올리면 링크 표시가 나타납니다.  
![][image3]

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/BEJ8AaUWCIA) 

## **2\. 하이퍼링크 열기**

1\) 링크가 지정된 객체를 선택합니다.  
2\) Ctrl \+ 마우스 왼쪽 버튼 클릭합니다.  
3\) 지정한 웹 브라우저, 프로그램, 전자우편 클라이언트 등이 실행됩니다.

## **3\. 하이퍼링크 편집 / 삭제**

1\) “HYPERLINK” 명령어를 실행합니다.  
2\) 링크가 걸린 객체를 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) 하이퍼링크 대화 상자가 나타납니다.  
5\) 링크 주소를 변경하거나, 삭제할 수 있습니다.  
6\) “확인”을 클릭하여 설정을 완료합니다.
