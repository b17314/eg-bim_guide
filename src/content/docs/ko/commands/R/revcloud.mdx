* 리비전 구름마크를 그리는 명령어입니다.  
* 리비전 구름마크를 그리거나 기존에 그려져있던 객체를 리비전 구름마크로 변환할 수 있습니다.

![][image1]

* 리비전 구름마크 그리기 옵션 키워드  
- Arc length: 구름마크 호의 최소/최대 길이를 설정합니다.  
- Entity: 객체를 리비전 구름마크로 변환합니다.  
- Style: 리비전 구름마크의 스타일을 설정합니다.

![][image2]

* 리비전 구름마크 스타일 옵션 키워드

![][image3]

* 리비전 구름마크 방향 옵션 키워드  
- Yes 키워드가 선택된 경우 현재 그려진 리비전 구름마크의 방향을 뒤집습니다.

## **1\. 리비전 구름마크를 그리는 방법**

1\) “REVCLOUD” 명령어를 실행합니다.  
2\) 필요한 경우 Arc length/Style 키워드를 선택하여 리비전 구름마크의 옵션을 설정합니다.  
3\) 화면 상에 마우스로 시작점을 지정합니다.  
3\) 화면 상에서 마우스를 움직이거나 클릭해 리비전 구름마크를 그립니다.  
3\) C를 눌러 리비전 구름마크를 닫거나 ENTER 키를 누릅니다.  
4\) 방향을 반전시킬지 옵션을 선택하고나면 리비전 구름마크 생성이 완료됩니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/zBPNkeqSmUA) 

## **2\. 객체를 리비전 구름마크로 변환하는 방법**

1\) “REVCLOUD” 명령어를 실행합니다.  
2\) Entity 키워드를 선택합니다.  
3\) 객체를 마우스로 선택합니다.  
4\) 방향을 반전할지 옵션을 선택하면 객체가 리비전 구름마크로 변환됩니다.
