* 현재 도면의 표시 데이터를 다시 계산하여 화면을 재생성하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “REZEN” 입력합니다.  
2\) 엔터 키를 누릅니다.  
3\) 현재 뷰가 다시 계산되어 화면이 새로 고쳐집니다.
