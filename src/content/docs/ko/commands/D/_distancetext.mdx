* 두 점 간의 거리(Length)를 자동으로 계산하여 텍스트로 표시하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “DISTANCETEXT” 입력합니다.  
2\) 명령창에 S 를 입력하면 설정 대화상자가 나타납니다.  
![][image2]  
3\) 대화상자에서 필요한 옵션을 선택합니다.  
3\) 명령창 안내에 따라 두 점(Point) 을 순서대로 지정합니다.

- 첫 번째 점: 거리 측정의 시작점  
- 두 번째 점: 거리 측정의 끝점

5\) 두 점 간의 직선 거리 가 명령창에 표시되며, 동시에 해당 위치에 텍스트 형태로 거리 값이 삽입됩니다.
