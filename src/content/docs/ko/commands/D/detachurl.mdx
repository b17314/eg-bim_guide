* 객체에 연결된 하이퍼링크를 삭제하여, 더 이상 해당 객체에서 링크가 열리지 않도록 하는 기능입니다.

![][image1]

## **1\. 객체 하이퍼링크 제거**

1\) “DETACHURL” 명령어를 실행합니다.  
2\) 하이퍼링크가 설정된 객체(선, 폴리선, 블록, 이미지 등)를 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) 선택한 객체에 포함된 모든 하이퍼링크가 제거됩니다.
