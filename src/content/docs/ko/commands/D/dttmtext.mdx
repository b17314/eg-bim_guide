* DTEXT(단일행 텍스트)를 MTEXT(다중행 텍스트) 로 변환하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “DTTMTEXT” 명령어를 실행합니다.  
2\) DTTMTEXT 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 변환할 단일행 텍스트 객체(DTEXT) 를 선택합니다.  
4\) 엔터 또는 스페이스 키를 누릅니다.  
5\) 선택된 텍스트가 하나의 MTEXT 객체로 변환됩니다.
