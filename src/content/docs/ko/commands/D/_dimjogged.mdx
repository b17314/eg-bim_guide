* 선택한 원 또는 호의 반지름을 측정하여 꺽어진 치수로 표시합니다.

## **1\. 꺽은선 반지름 치수 입력**

1\) DIMJOGGED 명령을 실행합니다.   
![][image1]  
2\) 도면에 반지름을 측정할 원 또는 호를 선택합니다.   
3\) 꺽어진 반지름 치수를 작성할 새 중심점을 지정합니다.  
![][image2]  
4\) 마우스를 움직여 원하는 치수 문자 위치에서 마우스를 클릭합니다.  
5\) 마우스를 움직여 원하는 치수선의 각도에서 마우스를 클릭합니다.  
![][image3]

## **2\. 꺽은선 반지름 치수 문자의 각도 변경**

1\) DIMJOGGED 명령을 실행하고, 도면에 반지름을 측정할 원 또는 호를 선택합니다.  
2\) Angle 옵션을 클릭 후 각도을 입력하면, 입력한 각도로 치수 문자가 기입 됩니다. (기본값은 0도)  
![][image4]  
![][image5]  
![][image6]

## **3\. 꺽은선 반지름 치수를 여러줄 문자로 입력하는 방법**

1\) DIMJOGGED 명령을 실행하고, 도면에 반지름을 측정할 원 또는 호를 선택합니다.    
2\) Mtext 옵션을 클릭하여 Text Editor 툴바가 나타나고 치수 문자가 편집 상태가 되면, 여러줄로 문자를 입력합니다.  
![][image7]  
![][image8]  
3\) Text Editor 툴바의 Done버튼(V)을 클릭하거나, 도면의 여백 부분을 클릭하여 편집을 종료합니다.  
![][image9]

## **4\. 지름 치수 문자를 변경하는 방법**

1\) DIMJOGGED 명령을 실행하고, 도면에 반지름을 측정할 원 또는 호를 선택합니다.    
2\) Text 옵션을 클릭하고, 변경할 문자열을 입력합니다.  
![][image10]  
3\) Enter 키를 누르고, 도면에서 지름 치수를 원하는 위치에 기입합니다.  
![][image11]
