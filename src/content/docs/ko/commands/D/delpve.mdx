## **1\. 폴리선 정점 삭제하기**

1\) “DELPVE” 명령어를 실행합니다.  
2\) 정점을 삭제할 폴리선 객체를 선택합니다.  
3\) 정점을 삭제할 점을 마우스로 클릭합니다.(더 이상 삭제할 정점이 없으면 명령어는 종료됩니다.)  
4\) 명령어를 종료하려면 “ESC”키를 입력합니다.
