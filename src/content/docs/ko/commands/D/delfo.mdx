* 도면에서 특정 조건에 해당하는 레이어의 객체를 삭제하는 기능입니다. 레이어 자체는 삭제하지 않고, 그 레이어에 속한 객체만 제거합니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “DELFO” 명령어를 입력합니다.  
2\) DELFO 대화상자에서 삭제할 레이어의 조건을 설정합니다.  
![][image2]  
3\) 설정 완료 후 확인 버튼을 클릭합니다.  
4\) 선택된 조건에 맞는 레이어 객체가 삭제됩니다.
