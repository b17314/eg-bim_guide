* 폴리선의 방향을 전환하는 명령어입니다.  
* 방향 전환 가능한 객체는 선, 폴리선, 호 입니다.  
* 객체의 시작점과 끝점이 변경됩니다.

![][image1]

## **1\. 선, 폴리선, 호 객체의 방향전환하기**

1\) “DIRECTIONREVERSE” 명령어를 실행합니다.  
2\) 도면에서 방향을 바꿀 객체(선, 폴리선, 호)를 선택합니다.  
3\) 방향 전환된 객체를 선택하여 속성창에서 확인합니다.
