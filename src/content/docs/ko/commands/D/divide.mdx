* 객체를 지정한 개수만큼 균등 분할하여 점 또는 블록을 자동 배치하는 기능입니다.

![][image1]

## **1\. 객체 균등 분할**

1\) “DIVIDE” 명령어를 실행합니다.  
2\) 균등하게 나눌 객체(선, 폴리선, 원, 아크 등) 를 선택합니다.  
3\) 엔터 또는 스페이스 키를 눌러 선택을 완료합니다.  
4\) 나눌 분할 개수를 입력합니다.  
5\) 엔터 또는 스페이스 키를 누릅니다.  
6\) 객체가 입력한 개수만큼 균등하게 분할되고, 점 또는 블록이 자동 배치됩니다.
