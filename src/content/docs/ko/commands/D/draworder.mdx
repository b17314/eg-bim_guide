![][image1]

## **1\. 객체 표시 순서 변경**

1\) “DRAWORDER” 명령어를 실행합니다.  
2\) 순서를 변경할 객체(선, 해치, 이미지, 블록 등)를 선택합니다.  
3\) 엔터 또는 스페이스 키를 눌러 선택을 완료합니다.  
4\) 다음 옵션 중 하나를 선택합니다:

- Front → 선택한 객체를 맨 앞으로 가져오기  
- Back → 선택한 객체를 맨 뒤로 보내기  
- Above objects → 특정 객체 위로 올리기  
- Under objects → 특정 객체 아래로 내리기
