* 선택한 객체를 잘라내기 하여 클립보드에 복사하고, 원본 객체는 도면에서 삭제하는 기능입니다.

![][image1]

## **1\. 객체 잘라내기**

1\) “CUTCLIP” 명령어를 실행합니다.  
2\) 잘라낼 객체(선, 폴리선, 원, 블록, 치수선 등)를 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) 선택한 객체가 클립보드에 복사되면서, 도면에서는 삭제됩니다.  
    참고 : PASTECLIP 명령어를 실행하여 도면에 붙여넣습니다.
