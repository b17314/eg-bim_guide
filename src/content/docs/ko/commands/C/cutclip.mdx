* 선택한 객체를 잘라내어 클립보드에 저장하는 명령어입니다.

![][image1]

## **1\. 객체 잘라내기**

1\) “CUTCLIP” 명령어를 실행합니다.  
2\) 잘라낼 객체를 선택합니다.  
3\) 엔터키를 누릅니다.  
4\) 선택한 객체가 도면에서 제거되고 클립보드에 저장됩니다.  
    참고 : 이후 PASTECLIP 명령어나 CTRL+V를 이용해 원하는 위치에 붙여넣을 수 있습니다.
