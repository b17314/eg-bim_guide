* 현재 뷰포트화면을 이미지로 캡처하여 클립보드 에 복사하는 기능입니다.  
* 복사된 이미지는 도면에 붙여넣거나, 외부 프로그램(Word, PowerPoint, Paint 등)에 붙여넣어 활용할 수 있습니다.

## **1\. 화면 이미지 캡처**

1\) 캡처할 객체를 선택한다.  
2\) “COPYCAPTUREDIMAGE” 명령어를 실행합니다.  
3\) 선택한 객체는 이미지(비트맵) 형식으로 클립보드에 저장됩니다.  
    참고 : 이후 PASTESPEC 명령어 또는 일반 붙여넣기(Ctrl+V)를 이용하여 이미지 붙여넣기  
    가능하다.
