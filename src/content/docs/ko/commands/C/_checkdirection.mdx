* 도면 내 폴리라인이나 해치의 방향을 확인하거나 교정할 때 사용하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “SCALEDISTANCE” 입력합니다.  
2\) 방향을 확인할 객체를 선택합니다.  
3\) 선택한 객체의 현재 방향이 객체에 표시됩니다.
