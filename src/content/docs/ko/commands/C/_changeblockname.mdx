* 도면에 정의된 블록의 이름을 변경하는 기능입니다. 기존 블록 이름을 새로운 이름으로 바꿀 수 있으며, 변경된 블록 이름은 도면 전체에 즉시 반영됩니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “CHANGEBLOCKNAME” 명령어를 입력합니다.  
2\) 이름을 변경할 블록을 선택합니다.  
2\) 명령창에서 변경할 새 블록 이름을 입력합니다.  
4\) 엔터 또는 스페이스 키를 눌러 적용합니다.  
5\) 도면 내 기존 블록 참조가 모두 새로운 이름으로 업데이트됩니다.
