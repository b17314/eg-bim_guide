* 선택한 객체를 사용자가 지정한 다른 레이어로 복사하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “COPYTOLAYER” 명령어를 입력합니다.  
2\) 엔터 키를 누른다.  
3\) 명령창에 S를 입력하면 대화상자가 나타납니다.  
![][image2]  
4\) 대화상자에서 원하는 레이어 이름을 입력합니다.  
5\) 복사할 객체를 선택합니다.  
6\) 복사 명령이 완료되면, 선택한 객체가 지정한 레이어로 복사되며, 원본 객체는 그대로 유지됩니니다.  
![][image3]
