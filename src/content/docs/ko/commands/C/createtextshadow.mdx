* 선택한 텍스트 객체에 외곽선또는 그림자 효과를 적용하여 시각적으로 강조하는 기능입니다.

![][image1]

## **1\. 문자 외곽선 생성하기**

1\) “CREATETEXTSHADOW” 명령어를 실행합니다.  
2\) “S” 를 입력하고 CREATETEXTSHADOW 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 확인을 누르고 그림자 효과를 적용할 텍스트 객체를 선택합니다.  
4\) 엔터 또는 스페이스 키를 누릅니다.  
5\) 선택한 텍스트에 그림자 효과가 적용됩니다.  
![][image3]
