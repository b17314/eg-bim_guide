* 선택한 객체를 기준점과 목표점을 지정하여 복사본을 생성하는 기능입니다.

![][image1]

## **1\. 객체 복사**

1\) “COPY” 또는 “CO” 명령어를 실행합니다.“COPY” 또는 “CO” / “CP” 명령어를 실행합니다.  
2\) 복사할 객체(선, 폴리선, 원, 블록, 치수선 등)를 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) 기준이 될 첫 번째 점(기준점) 을 지정합니다.  
5\) 복사할 두 번째 점(목표점) 을 지정합니다.  
    참고 : 필요 시 추가 목표점을 계속 지정하면, 여러 개의 복사본을 연속적으로 생성할 수 있습니다.  
6\) 엔터 또는 스페이스 키를 눌러 종료합니다.
