* 선택한 텍스트 객체의 내용을 대문자·소문자·첫 글자 대문자 등으로 변환하는 기능입니다.

![][image1]

## **1\. 대소문자 변경하기**

1\) “CONVERTTC” 명령어를 실행합니다.  
2\) CONVERTTC 대화상자에서 원하는 변환 방식을 선택합니다.  
![][image2]  
3\) 변환할 텍스트 객체를 선택합니다.  
4\) 엔터 또는 스페이스 키를 누르면 선택한 택스트의 대소문자가 지정한 방식으로 변환된다.  
![][image3]
