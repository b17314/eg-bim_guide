* 도면 상에서 횡단보도를 자동으로 생성하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “CROSSWALK” 를 입력합니다.  
2\) 설정에서 필요한 옵션션을 설정합니다.  
![][image2]  
3\) 도면에서 횡단보도의 시작점을 지정합니다.  
4\) 도면에서 횡단보도의 끝점을 지정합니다.  
5\) 엔터 키를 누르면 지정 구간에 횡단보도 패턴이 자동 생성됩니다.
