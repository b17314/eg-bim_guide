* 선택한 사각형 영역 또는 도형의 중심점으로 문자 객체를 자동 이동시키는 기능입니다.

![][image1]

## **1\. 문자 중심 이동하기**

1\) 명령창에 “CENTERREC” 명령어를 실행합니다.  
2\) 이동시킬 문자 객체를 선택합니다.  
3\) 중심을 기준으로 삼을 사각 영역을 선택합니다.  
4\) 확인하면 선택한 문자가 사각 영역의 중심 좌표로 자동 정렬됩니다.
