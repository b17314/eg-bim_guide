* 선택한 객체의 선 두께를 일괄적으로 변경합니다.

![][image1]

## **1\. 객체 색상 변경**

1\) “CHCOLOR” 명령어를 실행합니다.  
2\) 색상을 변경할 객체(선, 폴리선, 원, 사각형 등) 을 선택합니다.  
3\) 엔터를 눌러 선택을 완료합니다.  
4\) 적용할 색상 번호를 입력합니다.  
5\) 엔터 또는 스페이스 키를 누릅니다.  
6\) 선택한 객체의 색상이 입력한 번호에 해당하는 색으로 변경됩니다.  
![][image2]

## **2\. 색상 번호표**

| 번호 | 색상 이름 | 설명 |
| :---: | :---: | :---: |
| 0 | BYBLOCK | 객체가 속한 블록의 색상을 따름 |
| 1 | Red | 빨간색 |
| 2 | Yellow | 노란색 |
| 3 | Green | 초록색 |
| 4 | Cyan | 청록색 |
| 5 | Blue | 파란색 |
| 6 | Magenta | 마젠타색(자홍) |
| 7 | White / Black | 화면 배경색에 따라 흰색 도는 검정으로 표시 |
| 8 \~ 9 | Light Gray 계열 | 회색 계열 |
| 10 \~249 | 표준 ACI 팔레트 | 빨강\~파랑\~보라\~회색 등 점진적 색상 스펙트럼 |
| 250 | Gray 50% | 중간 회색 |
| 251 | Gray 75% | 밝은 회색 |
| 252 | Gray 87% | 더 밝은 회색 |
| 253 | Gray 93% | 매우 밝은 회색 |
| 254 | Gray 96% | 거의 흰색에 가까운 회색 |
| 255 | Gray 100% | 순수 흰색 |
| 256 | BYLAER | 객체가 속한 레이어의 색상을 따름름 |
