* 폴리선을 3D 폴리선으로, 또는 3D 폴리선을 폴리선으로 변환하는 명령어입니다.  
* 폴리선을 변환하고 원본 폴리선을 삭제하거나 변환된 폴리선의 레이어를 지정하여 변경하는 작업을 추가로 진행할 수 있습니다.

![][image1]

* 변환 후, 폴리선 형식: 선택된 폴리선/3D 폴리선이 변환될 타입을 지정합니다.  
* 변환 후, 원본 삭제: 체크된 경우 변환의 대상이 된 원본 객체를 삭제합니다. 체크되지 않은 경우 원본 객체를 삭제하지 않습니다.  
* 변환 후, 레이어 변경: 체크된 경우 변환된 객체의 레이어를 레이어 버튼에서 지정된 레이어로 변경합니다. 체크되지 않은 경우 변환된 객체의 레이어는 원본 객체과 동일합니다.

## **1\. 폴리선을 상호 변환하는 방법**

1\) “CPLINE” 명령어를 실행합니다.  
2\) 변환할 폴리선/3D 폴리선 객체를 선택하고 ENTER 키를 누릅니다.  
3\) 선택한 폴리선/3D 폴리선이 팝업에서 설정한 폴리선 형식으로 변환됩니다.
