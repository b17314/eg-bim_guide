* 사용자가 직접 정의한 좌표계를 생성, 관리, 또는 전환하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “UCS” 를 입력합니다.  
2\) 필요한 옵션을 선택하여 새로운 좌표계를 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| 3point (3점) | 새 원점과 X축, Y축 방향을 세 점으로 지정하여 UCS를 설정합니다. |
| Delete (삭제) | 저장된 UCS를 삭제합니다. |
| Entity (객체) | 특정 객체(선, 면 등)의 방향에 맞춰 UCS를 자동으로 설정합니다. |
| Origin (원점) | UCS의 원점(0,0,0)을 새 위치로 이동시킵니다. |
| Previous (이전) | 이전에 사용하던 UCS 상태로 되돌립니다. |
| Restore (복원) | 이름으로 저장된 UCS를 불러옵니다. |
| Save (저장) | 현재 UCS를 이름으로 저장합니다. |
| View (뷰) | 현재 뷰 방향(View Direction)을 기준으로 UCS를 설정합니다. |
| X / Y / Z (축 회전) | 선택한 축(X, Y, Z)을 기준으로 UCS를 회전시킵니다. |
| World (세계) | 기본 세계좌표계(WCS, World Coordinate System)로 복귀합니다. |

3\) 엔터 키를 누르면 선택한 기준으로 UCS가 변경됩니다.   
    참고 : 변경된 UCS는 이후 그리기, 복사, 회전 등의 작업 기준이 됩니다.
