* 도면 속성, 객체 속성, 시스템 변수 등의 값을 불러와 동적으로 표시되는 텍스트(필드)를 삽입하는 기능입니다.

![][image1]

## **1\. 필드 생성**

1\) “FIELD” 명령어를 실행합니다.  
2\) 나타나는 필드 대화상자에서 필요한 항목을 설정합니다.  
![][image2]

- 필드 카테고리 : 객체, 도면, 날짜/시간, 플롯, 사용자 정의 등 선택  
- 필드 이름 : 선택한 카테고리 안의 세부 항목 지정 (예: 작성자, 도면 이름, 객체 길이 등)  
- 포맷 : 표시 방식(소수점 자리수, 단위, 텍스트 형식 등) 지정

3\) 확인(OK)을 누르면 도면에 MTEXT 형태로 필드가 삽입됩니다.  
![][image3]
