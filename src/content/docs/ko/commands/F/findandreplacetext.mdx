* 도면 내 여러 객체의 텍스트를 검색 및 일괄 변경할 수 있는 기능입니다. 

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “FINDANDREPLACETEXT” 명령어를 실행합니다.  
2\) FINDANDREPLACETEXT 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 검색, 변경, 일괄 변경 버튼을 사용하여 문자열을 검색하거나 변경합니다.  
4\) 적용 버튼을 클릭하면 조건에 맞는 문자열이 변경되고, 도면에 반영됩니다.
