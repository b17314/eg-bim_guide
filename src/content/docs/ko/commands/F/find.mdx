* 도면 내의 특정 문자열을 검색하고, 필요 시 다른 문자열로 치환하는 기능입니다.

![][image1]

## **1\. 문자 찾기**

1\) 명령창에 “FIND” 명령어를 실행합니다.  
2\) 찾기/바꾸기 대화상자가 나타납니다.  
![][image2]  
3\) 옵션을 설정합니다.

- 찾을 문자열 입력  
- 바꿀 문자열 입력 (선택 사항)  
- 검색 범위 : 현재 도면 전체, 선택 영역, 특정 객체 유형 등 지정 가능  
- 검색 옵션 : 대소문자 구분, 전체 단어 일치 여부 등

4\) 찾기 버튼을 클릭하면 일치하는 항목이 표시 됩니다.
