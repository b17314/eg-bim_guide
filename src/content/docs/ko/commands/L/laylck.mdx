* 도면에서 선택한 객체가 속한 레이어를 잠그는 기능입니다. 잠긴 레이어에 속한 객체는 화면에 표시되지만, 편집(이동, 수정, 삭제 등)은 불가능합니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYLCK” 명령어를 입력합니다.  
2\) 도면에서 잠그고자 하는 객체를 선택합니다.  
3\) 해당 객체가 속한 레이어가 자동으로 잠금 처리됩니다.  
    참고 : 잠긴 레이어의 객체는 희미하게 표시되거나(투명도 적용), 편집이 불가능한 상태가 됩니다.
