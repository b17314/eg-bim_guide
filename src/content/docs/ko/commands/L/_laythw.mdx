* 도면 내에서 동결되어 있던 모든 레이어를 한 번에 해제 하는 명령입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYTHW” 명령어를 입력합니다.  
2\) 엔터 키를 누른다.  
3\) 도면 내의 모든 동결된 레이어가 해제 되며, 숨겨져 있던 객체들이 다시 화면에 나타난다.
