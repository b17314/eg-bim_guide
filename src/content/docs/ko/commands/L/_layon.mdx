* 도면에서 꺼져 있는 모든 레이어를 다시 켜서, 숨겨진 객체들이 화면에 표시되도록 하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYON” 명령어를 입력합니다.  
2\) 엔터 키를 누릅니다.  
3\) 현재 도면에서 Off 상태였던 모든 레이어가 다시 켜집니다.
