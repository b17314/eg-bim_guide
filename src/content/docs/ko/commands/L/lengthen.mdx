* 선택한 객체(선분, 호, 폴리선 등)의 길이나 각도를 조정하는 명령어입니다.

![][image1]

## **1\. 객체 길이 조정하기**

1\) LENGTHEN” 명령어를 실행합니다.  
2\) 다음 옵션 중 하나를 선택합니다.

- Dynamic(동적 조정)

마우스로 객체의 끝점을 직접 잡아 늘리거나 줄입니다. 실시간으로 길이 변화를 확인할 수 있습니다.

- Increment (증분)  
  현재 길이에 일정 값을 더하거나 빼서 늘리거나 줄입니다.  
  예: \+50 `→` 현재 길이에 50 단위 연장 /-20 `→` 현재 길이에서 20 단위 감소.   
- Percent (백분율)  
  객체의 전체 길이에 대해 % 단위로 늘리거나 줄입니다.  
  예: 120% `→` 전체 길이를 20% 늘림 / 50 % `→` 전체 길이를 절반으로 줄임.  
- Total (전체 길이 지정)  
  원하는 최종 길이를 직접 입력합니다.  
  예: 1000 입력 시, 해당 객체의 길이를 1000으로 맞춤.

3\) 객체를 선택합니다.  
4\) 입력한 옵션에 따라 길이가 조정됩니다.  
5\) 엔터키나 스페이스바를 눌러 종료합니다.
