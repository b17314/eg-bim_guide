* 잠긴 레이어에 속한 객체가 화면에서 얼마나 희미하게(페이드 효과) 표시될지를 제어하는 기능 입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYLOCKFADECTL” 명령어를 입력합니다.  
2\) 현재 설정된 값을 확인할 수 있습니다.

- 기본값은 보통 50 (50% 페이드)로 설정됨.

3\) 원하는 값을 입력합니다.

- 0 : 페이드 효과 없음 (잠긴 객체도 원래와 동일하게 표시)  
- 1 \~ 90 : 숫자가 클수록 객체가 희미하게 보임

4\) 엔터 또는 스페이스 키를 누르면 설정이 적용됩니다.
