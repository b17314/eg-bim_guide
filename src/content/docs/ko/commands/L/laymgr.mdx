* 도면에서 선택한 레이어를 다른 레이어와 병합(Merge) 하는 기능입니다.병합 대상 레이어의 모든 객체가 기준 레이어로 이동되며, 병합된 레이어는 삭제됩니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYMGR” 명령어를 입력합니다.  
2\) 병합할 대상 레이어의 객체를 선택한다.  
3\) 엔터를 누른 후, 기준 레이어의 객체를 선택한다.  
4\) 대상 레이어의 모든 객체가 기준 레이어로 이동된다.  
5\) 병합된 대상 레이어는 삭제된다.
