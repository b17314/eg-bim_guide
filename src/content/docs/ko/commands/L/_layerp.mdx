* 레이어 설정 변경 후, 직전의 레이어 상태로 되돌리는 기능 입니다. 즉, 가장 최근에 변경하기 전의 레이어 속성을 복원할 수 있습니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYERP” 명령어를 입력합니다.  
2\) 엔터를 누르면 직전 변경 전 레이어 상태가 복원됩니다.
