* 잠긴 레이어를 해제(Unlock) 하는 기능입니다. LAYLCK 명령어로 잠겨 편집 불가능했던 레이어를 다시 정상 편집 가능 상태로 되돌립니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYULK” 명령어를 입력합니다.  
2\) 도면에서 잠금 해제할 객체를 선택한다.  
3\) 선택한 객체가 속한 레이어의 잠금이 해제된다.
