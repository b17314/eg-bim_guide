* 현재 선 스타일을 가진 새로운 직선을 생성합니다.  
* 시작점을 화면 상에 직접 지정하거나 마지막 포인트를 시작점으로 지정할 수 있습니다.  
* 옵션을 지정하여 직선을 그릴 수 있고 UNDO로 직전에 실행되었던 직선 그리기 작업 취소가 가능합니다.  
* 끝점을 연달아 지정하면 직전에 그려진 직선의 끝점을 시작점으로 가지는 직선이 그려집니다.

- Angle: 직선의 각도 고정  
- Length: 직선의 길이 고정  
- Follow: 직전에 그려졌던 직선과 각도가 동일한 직선을 잇달아 그립니다. 길이만 변경 가능  
- Undo: 직전에 실행되었던 직선 그리기 작업을 취소합니다.

## **1\. 직선 그리기 방법**

1\) “LINE” 명령어를 실행합니다.  
2\) 마우스로 화면 상에 직선의 시작점을 지정합니다. (2가지 방법)  
&nbsp;&nbsp;&nbsp;&nbsp;2-1) 마우스로 화면 상에 직선의 시작점을 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;2-2) ENTER 키를 누르거나 “F”(Follow)키를 입력해 마지막 포인트를 시작점으로 지정합니다.  
3\) 마우스로 원하는 위치에 끝점을 지정합니다. (2가지 방법)  
&nbsp;&nbsp;&nbsp;&nbsp;3-1) 화면을 클릭해 끝점만을 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;3-2) “A”(Angle)를 입력하여 각도가 고정된 직선의 끝점을 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;3-3) “L”(Length)을 입력하여 길이가 고정된 직선의 끝점을 지정합니다.  
4\) 직선 그리기 작업을 계속하고 싶은 경우 마우스로 원하는 위치에 끝점을 지정합니다. (4가지 방법)  
&nbsp;&nbsp;&nbsp;&nbsp;4-1) 화면을 클릭해 끝점만을 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;4-2) “A”(Angle)를 입력하여 각도가 고정된 직선의 끝점을 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;4-3) “L”(Length)을 입력하여 길이가 고정된 직선의 끝점을 지정합니다.  
&nbsp;&nbsp;&nbsp;&nbsp;4-4) 직전에 그려진 직선이 있는 경우 “F”(Follow)를입력하여 직전 직선과각도가  
동일한 직선의 끝점을 지정합니다.  
5\) 직선 그리기 작업을 이전으로 되돌리고 싶은 경우 명령어창에서 “Undo” 키워드를 선택하거나 “U”(Undo)를 입력합니다.  
6\) 직선 그리기 작업을 종료하고자 하는 경우 ESC 또는 ENTER 키를 입력합니다.

## **2\. 닫힌 형태의 직선 그리기 방법**

1\) “LINE” 명령어를 실행합니다.  
2\) 최소 2개의 직선을 연달아 그립니다.  
3\) “C”(Close) 키워드를 입력하면 현재 실행 중인 “LINE” 명령어의 최초 시작점을 끝점으로 가지는 직선이 자동으로 그려지고 명령어가 종료됩니다.
