* 도면에서 사용하는 레이어를 생성·관리·편집하는 기능입니다. 레이어는 객체의 속성(색상, 선종류, 선가중치, 출력 여부 등)을 체계적으로 관리할 수 있게 하며, 도면 작업의 효율성과 가독성을 높여줍니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYER” 명령어를 입력합니다.  
2\) 레이어 속성 관리자 대화상자가 나타납니다.  
![][image2]  
3\) 다음과 같은 작업을 수행할 수 있다.

- 새 레이어 생성  
- 레이어 삭제  
- 레이어 이름 변경  
- 레이어 색상, 선종류, 선가중치 설정  
- 레이어 On/Off, Freeze/Thaw, Lock/Unlock 설정  
- 출력 여부(Plot) 설정

4\) 설정 완료 후 대화상자를 닫으면 변경 사항이 도면에 반영됩니다.
