* 도면의 길이 단위(Length Units) 형식을 설정하거나 변경하는 명령어입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LUNITS” 를 입력합니다.  
2\) 명령창에서 숫자 옵션을 선택합니다.

| 옵션 번호 | 단위 형식 | 설명 |
| :---: | :---: | ----- |
| 1 | 과학(Scientific) | 10의 지수 형태로 표시 (예: 1.23E+03) |
| 2 | 소수(Decimal) | 일반적인 실수 형태 (예: 123.4567) |
| 3 | 공학(Engineering) | 피트-인치 단위로 표시 (예: 12'-6.50") |
| 4 | 건축(Architectural) | 피트-인치 및 분수 단위로 표시 (예: 12'-6 1/2") |
| 5 | 분수(Fractional) | 분수 형태로 표시 (예: 6 1/2) |

3\) 엔터 키를 누르면 설정이 적용됩니다.
