* 도면에서 선택한 객체의 레이어를 현재 레이어 로 설정하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYER” 명령어를 입력합니다.  
2\) 현재 레이어로 지정하고 싶은 객체를 선택합니다.  
3\) 엔터를 누르면, 선택한 객체의 레이어가 현재 레이어로 전환됩니다.  
4\) 이후 작성되는 모든 새 객체는 지정된 레이어에 생성됩니다.
