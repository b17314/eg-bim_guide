* 도면에서 선택한 객체가 속한 레이어를 끄기 하여, 해당 레이어의 모든 객체를 화면에서 숨기는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “LAYOFF” 명령어를 입력합니다.  
2\) 끄고자 하는 레이어에 속한 객체를 선택합니다.  
3\) 엔터 키를 누르면 선택한 객체가 속한 레이어 전체가 Off 상태로 전환됩니다.   
    참고 : 해당 레이어에 속한 모든 객체가 화면에서 숨겨집니다.
