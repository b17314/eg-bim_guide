* 3차원 원구 를 그립니다.

## **1\. 반지름 지정하기**

1\) SPHERE 명령어를 실행합니다.  
2\) 구 의 중심이 될 좌표 선택 후 마우스 포인트 까지의 길이가 반지름인 원을 지정하면 해당 크기의 원구 가 그려집니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/kWyE5vnCYL4) 

## **2\. 지름 지정하기**

1\) SPHERE 명령어를 실행합니다.  
2\) 구 의 중심이 될 좌표 선택 후 명령어 창의 Diameter 옵션을 선택합니다.  
![][image1]  
3\) 마우스 포인트 까지의 길이가 지름인 원을 지정하면 해당크기의 원구가 생성됩니다.

[**<Icon name="youtube" class="emoji"/> 시연 영상 보기**](https://youtu.be/TslVx--_KTo)
