* 도면에서 사용할 문자 스타일을 생성, 수정, 관리하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “STYLE” 명령어를 실행합니다.  
2\) 문자 스타일 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 적용 버튼을 눌러 활성화합니다.  
4\) 이후 생성되는 텍스트는 지정된 스타일을 따릅니다.  
![][image3]
