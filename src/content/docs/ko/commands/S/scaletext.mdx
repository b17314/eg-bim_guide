* 선택한 문자 객체(DTEXT, MTEXT, 치수 문자 등)의 크기를 지정한 비율 또는 특정 높이에 맞게 조정 하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “SCALETEXT” 명령어를 실행합니다.  
2\) 크기를 조정할 문자 객체를 선택합니다.  
3\) 스케일 적용할 기준점을 중심(Center)으로 선택 합니다.  
4\) 현재 문자 크기에 곱할 배율을 입력 합니다.  
5\) 엔터 또는 스페이스 키를 누릅니다.  
6\) 선택한 문자의 크기가 조정됩니다.
