* 선택한 객체의 일부를 늘리거나 줄여 변형하는 기능입니다.

![][image1]

## **1\. 객체 늘리기 / 줄이기**

1\) “STRETCH” 또는 “S” 명령어를 실행합니다.  
2\) 교차 윈도우 선택으로 늘릴 부분을 포함하여 객체를 선택합니다.

- 선택된 객체의 일부 점만 포함해야 늘리기 동작이 적용됨

3\) 엔터를 눌러 선택을 완료합니다.  
4\) 기준점을 지정합니다.  
5\) 두 번째 점을 지정하여 늘리거나 줄일 방향과 거리를 설정합니다.  
6\) 선택된 부분이 지정한 벡터만큼 이동하며, 객체가 늘어나거나 줄어듭니다.
