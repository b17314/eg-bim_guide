* 사용자가 지정한 블록/속성블록/외부참조블록으로 해치를 넣을 수 있는 명령어업니다.  
* 명령어 팝업창의 블록 리스트 중에서 블록을 선택하고 해당 블록을 해치로 넣을 수 있습니다.

![][image1]

* 블록: 현재 도면의 블록 리스트  
* 속성블록: 현재 도면의 속성블록 리스트  
* 외부참조블록: 현재 도면의 외부참조블록 리스트  
* 리스트에서 블록을 선택하면 해당 블록이 오른쪽 검은색 미리보기 화면에 표출됩니다.

## **1\. 블록을 해치로 넣기**

1\) “SUPERHATCH” 명령어를 실행합니다.  
2\) 해치로 넣고자하는 블록을 선택합니다.  
3\) 확인 버튼을 누릅니다.  
4\) 해치 삽입점을 화면 상에 마우스로 지정합니다.  
5\) X축척 비율, Y축척 비율, 회전 각도를 입력합니다.  
6\) 경계선 내부 점을 화면 상에 마우스로 지정합니다.  
7\) ENTER 키를 누르면 지정한 경계선 내부에 선택한 블록이 해치로 넣어지고 명령어가 종료됩니다.
