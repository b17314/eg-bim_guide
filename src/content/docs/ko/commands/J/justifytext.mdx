* 선택한 문자 객체의 기준점(정렬 기준)을 변경하는 기능입니다.

![][image1]

## **1\. 기준점 변경하기**

1\) 명령창에 “JUSTIFYTEXT” 명령어를 실행합니다.  
2\) 정렬 기준을 변경할 문자 객체를 선택합니다.  
3\) 명령창에서 원하는 정렬 옵션을 지정합니다.

- 왼쪽(L)  
- 가운데(M)  
- 오른쪽(R)  
- 좌상단(TL) / 상단중앙(TC) / 우상단(TR)  
- 좌측중간(ML) / 중간중앙(MC) / 우측중간(MR)  
- 좌하단(BL) / 하단중앙(BC) / 우하단(BR)

4\) Enter를 누르면 선택한 문자의 기준점이 변경됩니다.
