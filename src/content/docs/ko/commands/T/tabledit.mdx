* 도면에 삽입된 표의 셀을 선택하여 직접 편집할 수 있는 기능입니다.

![][image1]

## **1\. 테이블 편집하기**

1\) 명령창에 “TABLEDIT`”` 명령어를 실행합니다.  
2\) 편집할 표의 셀을 클릭합니다.  
3\) 선택된 셀에 대해 도구막대가 나타나며, 셀 내부 내용을 편집을 할 수 있습니다.  
![][image2]

- 글꼴(Font) 변경  
- 문자 높이 및 간격  조정  
- 글자 스타일   
- 문자 색상 변경  
- 글머리 기호 및 번호 매기기  
- 정렬  
- 특수 문자 삽입  
- Import Text (외부 텍스트 불러오기)  
- AutoCAPS (자동 대문자 변환)

4\) Enter 또는 셀 밖을 클릭하면 변경 내용이 적용됩니다.
