* 선택한 객체를 기준으로 교차하는 부분을 잘라내어 필요 없는 부분을 제거 합니다.

![][image1]

## **1\. 객체 잘라내기**

1\) “TRIM” 또는 “TR” 명령어를 실행합니다.  
2\) 자를 기준이 되는 객체(절단선 역할)를 선택합니다.  
3\) 엔터키나 스페이스바를 누릅니다.  
4\) 잘라낼 부분(교차된 선의 필요 없는 부분)을 클릭합니다.  
5\) 엔터키나 스페이스바를 눌러 종료합니다.
