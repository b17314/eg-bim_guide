* 여러 개의 텍스트 객체(DTEXT, MTEXT)를 선택하여 하나의 문자열로 합치는 기능입니다.

![][image1]

## **1\. 문자 내용 합치기**

1)“TEXTJOIN” 명령어를 실행합니다.  
2\) “S” 를 입력하고 TEXTORDER 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 합칠 여러 텍스트 객체를 선택합니다.  
4\) 엔터 또는 스페이스 키를 누릅니다.  
5\) 선택된 텍스트들이 하나로 결합됩니다.  
![][image3]
