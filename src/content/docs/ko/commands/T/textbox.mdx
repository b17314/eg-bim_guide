* 선택한 텍스트 주위에 도형(박스, 원, 다각형 등) 을 자동으로 생성하여 강조하거나 배경 처리를 하는 기능입니다.

![][image1]

## **1\. 기본 사용 방법**

1\) “TEXTBOX” 명령어를 실행합니다.  
2\) “S” 를 입력하고 TEXTBOX 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 확인을 누르고 변환할 텍스트 객체를 선택합니다.   
4\) 엔터 또는 스페이스 키를 누릅니다.  
5\) 선택한 텍스트 주위에 설정된 모양의 도형이 생성됩니다.  
![][image3]
