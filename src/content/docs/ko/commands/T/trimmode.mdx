* 모깎기(FILLET)/모따기(CHAMFER) 작업 시 원본 선을 자동으로 자를지 여부를 결정하는 시스템 변수 명령어입니다.  
* 이 명령어는 원본 선을 자동으로 자를지 여부를 설정합니다.  
- TRIMMODE \= ON(1) : 유지 안함  
- TRIMMODE \= OFF(0) : 유지

![][image1]

## **1\. 모깎기 / 모따기 전 모서리 유지 안함**

1\) “TRIMMODE” 명령어를 실행하여 “1” 또는 “ON”을 입력합니다.  
2\) 확인하기 위해 “FILLET” 명령어를 실행합니다.  
3\) Radius(R) 키워드를 입력하여 값을 입력 후 “ENTER”를 입력합니다.(Radius가 0으로 설정되어 있으면 원호가 생성되지 않습니다.)  
4\) 교차되어있는 첫번째 선 객체를 선택합니다.  
5\) 두번째 선 객체를 선택하여 결과를 확인합니다.  
![][image2]

## **2\. 모깎기 / 모따기 전 모서리 유지**

1\) “TRIMMODE” 명령어를 실행하여 “0” 또는 “OFF”를 입력합니다.  
2\) 확인하기 위해 “FILLET” 명령어를 실행합니다.  
3\) Radius(R) 키워드를 입력하여 값을 입력 후 “ENTER”를 입력합니다.(Radius가 0으로 설정되어 있으면 원호가 생성되지 않습니다.)  
4\) 교차되어있는 첫번째 선 객체를 선택합니다.  
5\) 두번째 선 객체를 선택하여 결과를 확인합니다.
