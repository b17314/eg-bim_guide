* 여러 텍스트 객체에 포함된 특정 문자열을 한 번에 찾아 다른 문자로 일괄 변경하는 기능입니다.

![][image1]

## **1\. 문자 일괄 교체**

1\) “TEXTREPLACE” 명령어를 실행합니다.  
2\) 수정할 여러 텍스트 객체 선택합니다.  
3\) 지정 문자 입력 : 변경할 대상 문자 내용 입력합니다.  
4\) 대체 문자 입력 : 교체할 문자 내용입력합니다  
5\) 엔터 또는 스페이스 키를 누릅니다.  
6\) 선택된 모든 텍스트에서 해당 문자열이 일괄 교체됩니다.
