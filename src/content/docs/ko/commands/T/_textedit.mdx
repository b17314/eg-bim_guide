* 단일 줄 텍스트(DTEXT) 또는 여러 줄 텍스트(MTEXT) 객체를 편집하는 기능입니다.

![][image1]

## **1\. 특수문자 입력**

1\) “TEXTEDIT” 명령어를 실행합니다.  
2\) 편집할 텍스트 객체를 선택합니다.  
3\) 수정이 완료되면 Enter 키를 눌러 편집을 종료합니다.
