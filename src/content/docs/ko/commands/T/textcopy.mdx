* 선택한 텍스트 객체의 내용을 복사하여 다른 위치나 객체에 붙여넣는 기능입니다.

![][image1]

## **1\. 텍스트 복사**

1\) “TEXTCOPY” 명령어를 실행합니다.  
2\) 복사할 원본 텍스트 객체를 선택합니다.  
3\) 내용을 붙여넣을 대상 텍스트 객체를 선택합니다.  
4\) 엔터 또는 스페이스 키를 누릅니다.  
5\) 선택한 텍스트 객체의 내용이 복사됩니다.
