* 선택한 숫자 텍스트에 천 단위 구분기호(,)를 자동으로 삽입하여 표기 형식을 변경하는 기능입니다.

![][image1]

## **1\. 천 단위 표기**

1\) “THPOINT” 명령어를 실행합니다.  
2\) 천 단위 표기를 적용할 숫자 텍스트 객체를 선택합니다.  
3\) 명령어가 실행되면 선택한 숫자에 자동으로 천 단위 구분 쉼표(,) 가 추가됩니다..

- 예: 1000 → 1,000
