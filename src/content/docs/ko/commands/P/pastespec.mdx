* 클립보드에 복사한 데이터를 특정 형식(비트맵, Script Text 등)으로 선택하여 현재 도면에 붙여넣는 기능입니다.

![][image1]

## **1\. 객체 블록 붙여넣기**

1\) “PASTEBLOCK” 명령어를 실행합니다.   
    참고 : 이전에 COPYCLIP 명령어로 클립보드에 객체가 복사되어 있어야 합니다.  
2\) 붙여넣기 형식의 선택 상자가 나타난다.  
2\) 붙여넣을 기준점을 지정합니다.  
3\) 복사된 객체가 자동으로 임시 블록으로 변환되어 붙여넣어집니다.
