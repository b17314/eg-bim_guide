![][image1]

## **1\. 객체 속성 확인 및 수정**

1\) “PROPERTIES” 명령어를 실행합니다.  
2\) 속성을 확인할 객체(선, 폴리선, 원, 사각형, 블록 등) 을 선택합니다.  
3\) 화면 오른쪽 또는 별도의 속성창이 나타납니다.  
4\) 변경하고자 하는 항목을 선택한 뒤 값을 직접 입력하거나 선택합니다.   
    변경 사항은 즉시 객체에 적용 됩니다.
