* 도면의 표시 배율을 변경하지 않고, 화면에서 보이는 위치 만 이동시키는 명령입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “ZOOM” 명령어를 입력합니다.  
2\) 엔터 키를 누릅니다.  
3\) 커서가 ‘손 모양’으로 바뀌며, 마우스로 클릭 후 드래그하여 화면을 이동합니다.  
4\) 이동이 완료되면 Enter 또는 ESC 키로 명령을 종료합니다.  
    참고 : 마우스 휠을 눌러서 드래그하면 PAN 명령과 동일한 효과를 즉시 사용할 수 있습니다.
