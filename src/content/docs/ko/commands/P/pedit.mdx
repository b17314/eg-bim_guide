* 선택한 폴리선을 편집하는 명령어입니다.  
* 폴리선을 여러개 혹은 1개 선택하고 정점, 닫기 여부, 너비 등 폴리선의 속성을 편집할 수 있습니다.

![][image1]

* 폴리선 선택 옵션 키워드  
- Multiple: 여러개의 폴리선을 동시에 동일한 속성으로 편집합니다.

![][image2]  
![][image3]

* 폴리선 속성 편집 옵션 키워드  
- Edit Vertices: 폴리선의 꼭짓점들을 편집합니다.  
- Close: 폴리선을 닫힌 형태로 편집합니다.  
- Open: 폴리선을 열린 형태로 편집합니다.  
- Decurve: 폴리선의 곡선을 비곡선화 시킵니다.  
- Fit: 폴리선에 있는 꼭짓점들을 통과하는 곡선 폴리선으로 편집됩니다.  
- Join: 양 끝이 연결된 각각의 폴리선들을 하나의 폴리선으로 결합합니다.

&nbsp;&nbsp;&nbsp;일반 직선을 선택하는 경우, 폴리선으로 변환하여 다른 직선과의 결합도 가능합니다.  
&nbsp;&nbsp;&nbsp;(결합 결과 \= 폴리선선)

- Linetype-mode: 꼭짓점마다 선스타일을 다시 시작할지 여부를 설정합니다.  
- Reverse: 폴리선을 반전시킵니다.  
- Spline: 폴리선을 spline-fit하게 만듭니다.  
- Taper: 폴리선의 양끝 너비를 다르게 설정합니다.  
- Width: 폴리선의 너비를 설정정합니다.  
- Undo: 작업 취소  
- exit: 편집 종료

![][image4]

* 폴리선 꼭짓점 편집 옵션 키워드  
- Next: 다음 꼭짓점을 편집합니다.  
- Previous: 이전 꼭짓점을 편집합니다.  
- Angle: 다음 꼭짓점으로 이어지는 선의 기울기를 편집합니다.  
- Break: 다른 꼭짓점을 선택하여 해당 꼭짓점과 연결된 선을 끊습니다.  
- Insert: 현재 선택된 꼭짓점과 연결된 새 꼭짓점을 생성합니다.  
- Move: 현재 꼭짓점을 이동시킵니다.  
- Regen: 꼭짓점을 재생성합니다.  
- Select: 화면 상에서 마우스로 꼭짓점을 선택하여 편집할 꼭짓점으로 가져옵니다.  
- Straighten: 꼭짓점을 선택하고 해당 꼭짓점과 연결된 선에 곡선이 있는 경우 직선으로 편집합니다.  
- Tangent: 꼭짓점이 가질 각도를 설정합니다.  
- Width: 현재 꼭짓점에서 다음 꼭짓점을 연결하는 선의 너비를 편집합니다.  
- exit: 꼭짓점 편집을 종료합니다. 폴리선 속성 편집 상태로 돌아갑니다.

## **1\. 폴리선을 수정하는 방법**

1\) “PEDIT” 명령어를 실행합니다.  
2\) 폴리선을 선택합니다.  
3\) 편집하고자 하는 속성을 선택하여 편집을 진행합니다.  
4\) Undo 키워드를 입력하면 직전에 진행되었던 작업이 취소됩니다.  
5\) exit 키워드/Esc 키/Enter 키를 눌러 편집을 종료합니다.
