* 블록의 삽입 기준점을 지정하거나 변경하는 기능입니다.  
* 블록을 도면에 삽입할 때, 지정된 삽입점을 기준으로 배치되므로 정확한 위치 제어가 가능합니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “BINSERTIONPOINT” 명령어를 입력합니다.  
2\) 기준점을 지정할 블록을 선택합니다.  
3\) 기준점을 새로 지정할 위치를 마우스로 클릭하거나 좌표값을 입력합니다.  
4\) 지정 완료 후 블록의 삽입 기준점이 변경됩니다.
