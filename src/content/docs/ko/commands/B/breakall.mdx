* 선택한 객체의 모든 교차점 을 한 번에 끊어내내는 기능입니다.

![][image1]

## **1\. 객체 끊어내기**

1\) “BREAKALL” 명령어를 실행합니다.  
2\) 교차하는 객체(선, 폴리선, 원, 호 등)를 선택합니다.  
3\) 선택한 객체는 교차점 기준으로 모두 분리됩니다.
