* 도면 내 블록 정의를 편집하기 위해 블록 편집기를 여는 기능입니다.  
* BEDIT를 실행하면 특정 블록을 선택하여 내부 객체를 수정할 수 있으며, 블록 편집 후 저장 시 도면 내 모든 참조가 동시에 업데이트됩니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “BEDIT” 명령어를 입력합니다.  
2\) BEDIT 대화상자가 나타납니다. 이때 도면 내 존재하는 블록 목록이 표시됩니다.  
![][image2]  
3\) 편집할 블록을 선택합니다.  
4\) OK 버튼을 클릭하면 블록 편집기로 전환됩니다.  
![][image3]  
5\) 편집이 끝나면 BCLOSE 명령어를 실행하고, 변경 사항을 저장하면 도면 내 블록이 업데이트됩니다.  
![][image4]
