* 도면에 그려진 여러 객체를 하나의 블록으로 정의하여, 이후 재사용할 수 있게 만드는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “BLOCK” 명령어를 실행합니다.   
2\) BLOCK 대화상자에서 ADD 버튼을 클릭합니다.  
![][image2]  
3\) 원하는 블록을 선택 후 삽입 위치를 지정하면 도면에 블록이 배치됩니다.
