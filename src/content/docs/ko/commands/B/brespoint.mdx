* 객체 위에서 지정한 점 위치를 기준으로 객체를 끊어내는 기능입니다

![][image1]

## **1\. 객체 끊어내기**

1\) “BRESPOINT” 명령어를 실행합니다.  
2\) 객체(선, 폴리선, 사각형 등)를 선택합니다.  
3\) 객체 위에서 끊을 지점을 지정합니다.  
4\) 선택한 객체는 지정한 점을 기준으로 나누어 집니다.
